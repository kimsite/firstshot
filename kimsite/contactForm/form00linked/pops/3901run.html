The entirity of the mysqli query process is assign to this variable so that this variable can be used to see if the query ran properly or not. If it runs, everything gets put into the database and a "thank You" etc is printed out on the screen to acknowledge this fact. If it DOES'T run properly, a notice to this effect is printed out along with a call to some de-bugging functions. In either eventuality, this variable is the one that gets checked.
