PHP has the capability for you to define and use your own functions.

The syntax is:

function function_name () {

the code

}

FUNCTION names (unlike VARIABLE names) are case-INsensitive

Self-defined functions can take any number of arguments (also called "parameters"). The ORDER is important.

To allow for arguments, add variiables to a function's definition:

function function_name($var_one, $var_two, $var_etc) {

the code

}

Once the function is defined, you can call it just like any other. (ie, you dont need to use the "function" part again, just the function name.)
