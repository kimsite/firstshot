If the variable "$run" is empty, and thus the SQL query failed to run, this is a (private) de-bugging set of two messages to that effect.

First, the error message(s) from MySQL is/are printed out (using the "mysqli_error()" function). Second, the query as originally writen (in script line 38) is printed out- to help with de-bugging.

Be careful as to who gets to see these error messages. They could give away information that would be valuable to a hacker etc.
