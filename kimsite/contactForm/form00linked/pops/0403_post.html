"$_POST" is one of PHP's predefined arrays (see script line 07), It is called a superglobal variable or a superglobal array.

$_POST stores all of the data sent to a PHP script from an HTML form that uses the "post" method.

You MUST capitalize this variable! PHP is case-sensitive for VARIABLE names (but not function names) and this is predefined as upper-case.
