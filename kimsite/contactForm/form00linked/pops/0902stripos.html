The foreach loop will access each item in the array $very_bad one at a time (by assigning it to the $oneatatime variable). Within the loop, the stripos() function will check if the item is in the string provided to this function as $submitted_value. (You want the results of the search to be "false" because that means that there aren't any "bad" strings presesnt.)

The first time that any of these items is found in the submitted value, the function will return an empty string and terminate (functions automaticly stop executing when they hit a return).

The stripos() function preforms a case-INsensitive search.
