This line opens a connection to a particular MySQL database. It calls up and runs an external file (that you prepare before hand- see pg 226) that has all the information in it to connect. You need to connect to the database at a point in a script BEFORE that script needs to input or use any of the data in the database.

"mysqli" stands for "My Structured Query Language Improved"
