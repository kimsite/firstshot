"isset()" is a funtion which tests if a variable has a value (other than "NULL". That is a value of "0". "FALSE", or an empty string).

For TEXT (ie in text inputs in forms etc) use the "empty()" function (which tests whether a variable has any value set [ "0", "FALSE", an empty string, OR "NULL"]).(see script line 17)
