The "mysqli_query()" function sends the SQL command (as assigned to "$q" on the previous line) to the MySQL database (as defined in "$dbc" as supplied by mysqli_connect() ).

The whole query is assigned to a variable ("$run") and checked as to whether it ran properly or not (dealt with in the following lines).

(The "@" symbol just suppresses error reporting during this step. Any errors that may occur are dealt with by the following lines.)
