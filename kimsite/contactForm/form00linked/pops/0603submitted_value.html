This is the Self-defined "spam_scrubber" function's single "argument". It is a variable and will take a string as the argument. (The string will be the user suplied text in each of the HTML form fields below.)

The text inputted in the form will be "scrubbed" of certain specific terms if they are present (in this case, terms that are commonly used to send spam) and then "newline" characters will be dis-allowed. A string that passes these processes will be given a new variable name and then THAT variable name will be used in the rest of the script (ie, "$scrubbed").
