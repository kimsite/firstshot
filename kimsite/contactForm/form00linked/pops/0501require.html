PHP has four functions FOR USING EXTERNAL FILES:

  • include()
  • include_once()
  • require()
  • require_once()

They would look like:

  • include_once('filename.php'); or
  • require('/path/to/filename.php'); etc

Using any one of these functions has the end result of taking all the content of the included file and dropping it into the parent script (the one calling the function) at that juncture.

The include() and require() functions are exactly the same when working properly but behave diferently when they fail:

  • include(): prints warning to web browser - script still runs
  • require(): prints warning to web browser - script stops running

The "_once" part guarantees that the file is included only once regardless
