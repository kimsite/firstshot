This says: "if" the "$errors" array is "empty" then do...

If this runs, it means that there were no errors to put in the $errors array and so all of the text fields in the form have something (cleaned up) in them. The (pristine) data is now ready to put in the database (next line).

If this does NOT run (ie if there WERE errors put into the "$errors" array) then this line returns as "FALSE" and the script jumps down to the "} else {" on line 49. (And prints out the errors).
