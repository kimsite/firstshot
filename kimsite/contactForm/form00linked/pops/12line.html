This is still part of the self-defined function "spam_scrubber".

This line replaces any newline characters with spaces. Newline characters may or may not be problematic. A newline character is required to send spam (or else you can't create the proper header) but will also appear if a user just hits Enter while typing in a textarea box. For this reason, any found newlines will just be replaced by a space.

The str_replace() function looks through the value in the third argument, and replaces any occurences of the characters in the first argument with the character(s) in the second argument.

This function can take strings OR arrays for any of its three arguments

So this line of code assigns to the $submitted_value variable its original value, with any newline characters replaced by a single space.
