mysqli_connect.php is an external file that contain the information needed to connect to a mysql database.

This file should be stored SECURELY because it contains sensitive MySQL access information- place it in the directory immediately above or otherwise outside of the Web directory. This way the file will not be accessible from a Web browser. (see pg 229)

This external file also establishes the $dbc variable used several places later in the script!
