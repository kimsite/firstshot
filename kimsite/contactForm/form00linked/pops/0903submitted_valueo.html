The foreach loop (script line 08) will access each item in the array $very_bad one at a time (by assigning it to the $oneatatime variable- also on script line 08). Within the loop, the stripos() function will check if the bad item is in the string provided to this function as $submitted_value. You want the results of each search to be "false" because that means that there aren't any "bad" strings presesnt. If the result is NOT false (ie true) that means that there is a "bad" string present and so an "empty string" is returned (and so the database only recieves a blank string instead of something bad).

The first time that any of these items is found in the submitted value, the function will return an empty string and terminate (functions automaticly stop executing when they hit a return).
