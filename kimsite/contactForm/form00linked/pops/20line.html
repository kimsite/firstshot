The user-supplied 'first_name' (or other variable) that is in the $scrubbed array (meaning that the variable has had any spamming related characters removed) has any HTML or PHP tags stripped off by the strip_tags() function and then the resulting string gets "trimmed" of any unnecessary spaces and is sent through the mysqli security function. The result is assigned to a new variable ($fn in this case) that is clean and ready to go into the database.
